# Bubble Sort

Python 實現的 Bubble Sort 排序演算法。

## 說明

Bubble Sort 是一種簡單的排序演算法，透過重複交換相鄰元素將最大值逐步移至末端。

## 使用方法

```python
from bubble_sort import bubble_sort

data = [5, 3, 8, 4, 2]
sorted_data = bubble_sort(data)
print(sorted_data)  # [2, 3, 4, 5, 8]
```

## 執行

```bash
python3 bubble_sort/bubble_sort.py
```
